## Variant Annotation & Phylogenetic Analysis
Functional annotation of filtered SNPs using SnpEff, followed by calculation of four-fold degenerate transversion (4DTV) sites via custom script `calc_4dTv_in_eff_vcf.py`. Results were aggregated across all 12 chromosomes, converted to PHYLIP format using `vcf2phylip.py`, and used to construct a maximum-likelihood phylogenetic tree with IQ-TREE (GTR model, 1,000 ultrafast bootstraps).
