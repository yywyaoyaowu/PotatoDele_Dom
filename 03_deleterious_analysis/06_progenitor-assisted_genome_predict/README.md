## Progenitor-Assisted Genome Prediction
Constructs a genome-wide deleterious mutation burden matrix by integrating GERP conservation scores with genotype data across 100-kb sliding windows (excluding selective sweeps). The script calculates group-specific minima (Landrace, Candolleanum) and exports BED files identifying windows where the Candolleanum burden is lower than that of inbreds (A626 and E463).
