# Identification of deleterious mutations and burden

01 distance: This code is employed to compute the genetic distance matrix among samples.

02 frequency: This code is employed to compute allele frequencies across different populations, and to determine both the minor allele frequency (MAF) and the derived allele frequency (DAF). 

03 diffRegionDele: This code is employed to determine the number of deleterious variants across different genomic regions—including CDS, intron, UTR, UpDown5K, promoter, and GeneticRegion—as well as under varying GERP threshold conditions (GERP ≥ 2, GERP ≥ 2.75 and GERP ≥ 3.5). 

04 burden: This code is employed to determine the deleterious allele frequencies, deleterious genotypes, and the number of deleterious genotypes per individual across both the CND and Landrace panels.

05 rmBothMajor: This code is applied to perform Fisher's exact tests and to implement partial filtering of deleterious variants. 

06_progenitor-assisted_genome_predict: Progenitor-Assisted Genome Prediction
