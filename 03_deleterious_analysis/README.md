# Identification of deleterious mutations and burden

01 distance: This code is employed to compute the genetic distance matrix among samples. The input consists of three files: a whole genome VCF file named DP4 100.Q10.030.MR0..maf.0001.recode.vct, a VCF file for selective region named dom.Filter.000tvcf gz, and a VCF file for non-selective regions (control) named NotDom.Fitero.0001vcf gz.

02 frequency: This code is employed to compute allele frequencies across different populations, and to determine both the minor allele frequency (MAF) and the derived allele frequency (DAF). The input file, SolMsa 100Species_chr{01.12).fa BiAllelefreq.bed, contains positional information across different chromosomes, along with the MajorAlele and its corresponding count (MajorAlleleCount) in 100 Solanaceae species at each position. TheDerived Allele Frequency (DAF)s determined as follows: if the MaiorAllelecount in 100 Solanaceae species is greater than or equal to threethe allele differing from it is designated as the derived allele; if the MajorAllelecount is less than three, the Major Allele observed in the CND population is considered the ancestral allele, and the allele differing from it is defined as the derived allele

03 diffRegionDele: This code is employed to determine the number of deleterious variants across different genomic regions—including CDS, intron, UTR, UpDown5K, promoter, and GeneticRegion—as well as under varying GERP threshold conditions (GERP ≥ 2, GERP ≥ 2.75 and GERP ≥ 3.5). The input files, snpEf. anno. info. bed.can and snpEff.anno.info. bed. lan, contain chromosome, position and SNP annotation information. The gerp file is generated by intersecting the original input files with sites where the GERP score is greater than or equal to two, and this file additionally includes the corresponding GERP scores.

04 burden: This code is employed to determine the deleterious allele frequencies, deleterious genotypes, and the number of deleterious genotypes per individual across both the CND and Landrace panels. The input file Sol msa ConstraintedGERP2 withDepth withMajorAllele_ chr{01.12).xt contains sites with GERP scores greater than or equal to two, along with the MajorAllele and MajorAllele Count information for these sites across 100 Solanaceae species. The Inbred_list includes only A6-26 and E4-63.

05 rmBothMajor: This code is applied to perform Fisher's exact tests and to implement partial filtering of deleterious variants. The filtered deleterious variant files can subsequently be used to calculate deleterious allele frequencies in the inbred lines A6-26 and E4-63. The input file, Allchrs. DP4_100.GQ10.Q30.MR0.5.maf0.0001 SolMsaNonMajorDele_ Genotype_GERP2.75.txt.fiter.txt, is generated by integrating the files from chromosomes 1to 12 computed in the previous step. In this file, a genotype value of  indicates anomozygous non-deleterious allele, a value of 1indicates a heterozygous deleterious allele, and a value of 2 indicates a homozygous deleterious allele.
