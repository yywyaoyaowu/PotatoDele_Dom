# Identification of deleterious mutations and burden

01_distance: 
The input consists of three files: a whole genome VCF file named DP4_100.GQ10.Q30.MR0.5.maf0.0001.recode.vcf, a VCF file for selective region named dom.Filter0.0001.vcf.gz, and a VCF file for non-selective regions (control) named NotDom.Filter0.0001.vcf.gz.

02_frequency:
The input file, SolMsa_100Species_chr{01..12}.fa_BiAllelefreq.bed, contains positional information across different chromosomes, along with the MajorAllele and its corresponding count (MajorAlleleCount) in 100 Solanaceae species at each position. The Derived Allele Frequency (DAF) is determined as follows: if the MajorAlleleCount in 100 Solanaceae species is greater than or equal to three, the allele differing from it is designated as the derived allele; if the MajorAlleleCount is less than three, the Major Allele observed in the CND population is considered the ancestral allele, and the allele differing from it is defined as the derived allele.

03_diffRegionDele:
The input files, snpEff.anno.info.bed.can and snpEff.anno.info.bed.lan, contain chromosome, position and SNP annotation information. The gerp_file is generated by intersecting the original input files with sites where the GERP score is greater than or equal to two, and this file additionally includes the corresponding GERP scores.

04_burden:
The input file Sol_msa_ConstraintedGERP2_withDepth_withMajorAllele_chr{01..12}.txt contains sites with GERP scores greater than or equal to two, along with the MajorAllele and MajorAlleleCount information for these sites across 100 Solanaceae species. The Inbred_list includes only A6-26 and E4-63.

05_rmBothMajor:
The input file, Allchrs.DP4_100.GQ10.Q30.MR0.5.maf0.0001_SolMsaNonMajorDele_Genotype_GERP2.75.txt.filter.txt, is generated by integrating the files from chromosomes 1 to 12 computed in the previous step. In this file, a genotype value of 0 indicates a homozygous non-deleterious allele, a value of 1 indicates a heterozygous deleterious allele, and a value of 2 indicates a homozygous deleterious allele.
